🎲 Maxi Yatzy
Ett komplett Maxi Yatzy-spel (6 tärningar) som körs direkt i webbläsaren – inga installationer, inget bygge, bara en enda HTML-fil.
Spela här: https://toltjoka.github.io
Funktioner
	•	Spelas med 6 tärningar och tre kast per omgång, enligt riktiga Maxiyatzy-regler
	•	Alla kombinationer, inklusive de tre “extra” slagen: Hus (3+3), Torn (2+4) och Full stege (1–6)
	•	Bonus på 100 poäng vid minst 84 i övre sektionen, samt 100 poäng för Yatzy
	•	Sparade kast – använder du inte alla tre kast en omgång sparas resten till nästa omgång (visas som guldpluppar bredvid spelarens namn)
	•	Lägg till eller ta bort spelare fritt, från 1 till hur många som helst
	•	Highscore-lista som sparas i webbläsaren mellan omgångar
	•	Responsiv design, fungerar på mobil
Så spelar du
	1.	Skriv in namn på spelarna (lägg till fler med + Lägg till spelare)
	2.	Klicka Starta spel
	3.	Klicka KASTA TÄRNINGAR – klicka på enskilda tärningar för att hålla dem mellan kasten
	4.	När du är nöjd, klicka på den rad i protokollet du vill skriva poäng på
	5.	Spelet fortsätter tills alla spelare har fyllt i alla kombinationer
Teknik
Ren HTML, CSS och JavaScript i en enda fil (index.html) – inga beroenden, inget byggsteg. Highscore sparas lokalt hos varje spelare via localStorage, så listan är per enhet/webbläsare och delas inte automatiskt mellan olika besökare.
Publicera själv via GitHub Pages
	1.	Lägg filen i ett repo som index.html (i repots rot)
	2.	Gå till Settings → Pages
	3.	Under Build and deployment, välj Deploy from a branch
	4.	Välj din branch (t.ex. main) och mappen / (root)
	5.	Spara – sidan publiceras inom någon minut på https://dittanvändarnamn.github.io/reponamn/
